## SHELBY FC | Website laravel

Utilizar os comandos:
```s
composer update
```

Para criar as tabelas:
```s
php artisan migrate
```

para inserir os dados default nas tabelas | users e paises
```s
php artisan db:seed   
```

```s
npm install & npm run build
```


// inicia server laravel
```s
php artisan serve
```
